💀 混接的核心危險：三重致命
了解這三層，你才能理解為何混接比直接短路還危險
1 外殼帶電 — 看不見的殺手
PE 正常狀態電流 = 0。混接後 PE 上有完整回路電流（幾安培）。 PE 接著設備金屬外殼 → 所有外殼帶電。 同一條 PE 匯流排串起的：電腦機箱、鐵製桌腳、水管、鐵門——全部帶電。 人碰到任何一個金屬面 → 觸電。
2 RCBO 保護完全失效
RCBO 的邏輯：量 L 電流進去多少，N 電流回來多少，差值 = 漏電量。 混接後 PE 當 N 用 → L 電流進去多少，PE（當 N）電流回來多少。差值 = 0。 RCBO 永遠偵測不到異常，永遠不會跳脫。
3 危險狀態長期持續 — 最可怕
設備能正常運作 → 沒有人去查 → 趕工的師傅走了、工地驗收通過了。 危險狀態可能安靜持續 數個月到數年，直到有人碰了外殼才爆發。 這才是混接比短路更危險的原因：短路會立刻跳電，混接不會。
🔍 三種常見混接情境
每種情境都真實發生過，都有人因此死亡
極危險 情境一：直接拿 PE 當 N 回路用
做法：把 PE 線接到原本應該接 N 線的位置
電流路徑：L → 負載 → PE 導線 → 接地棒 → 大地 → 台電接地棒 → 中性點 → 回電源
結果：PE 上有完整電流 → 所有外殼帶電 → RCBO 差值 = 0 → 不跳脫
設備能動，外殼帶電，RCBO 失效，隨時出人命
極危險 情境二：N 線斷了，用 PE 跳接補（最常見！）
場景：施工趕工，N 線某處斷掉，師傅拿一條線把 N 斷點接到 PE 匯流排
為何設備還能動：電流找到了另一條路（PE → 大地）
三層危險： 外殼帶電（PE 有電流）+ RCBO 失效（差值 = 0）+ 長期持續（設備能動沒人查）
正確做法：找到 N 線斷點，重新接好或換新線。PE 只接外殼。
極危險 情境三：末端把 N 和 PE 短路接在一起
做法：在插座或設備端，把 N 和 PE 端子直接短接
結果：N 線電流流進 PE → 外殼帶電 + RCBO 失效
違反 CNS 和 IEC 60364，任何理由都不可接受
✅ 什麼叫「合法的 N/PE 合一」？
TN-C-S 系統 — 電源側合一合法，末端分開是鐵律
TN-C-S 系統定義
從電源到進戶主配電盤（MDB）這段，N 線和 PE 線合為一條「PEN 線」。 進入建築後，在 MDB 強制分離成獨立 N 線和 PE 線。分離後永不再合接。
✅ 合法條件（缺一不可）
• PEN 線截面積 ≥ 10mm²（銅）
• 分離只在 MDB 做，只做一次
• 分離後 N/PE 用不同顏色標示
• 末端設備 N 和 PE 永不短路
• 符合 CNS 及 IEC 60364
比較 TN-C-S 合法 末端混接（違規）
位置電源到 MDB（電源側）末端插座或設備（負載側）
目的系統設計節省線材趕工省事
分離MDB 後嚴格分離永遠混在一起
安全性符合規範，安全外殼帶電，致命
🎯 一句話記住
電源側合一（節省線材）→ 合法。負載側合一（偷懶趕工）→ 犯罪。
🔧 現場量測與判斷
帶著三用電錶和鉗型電流表，這五個量測告訴你現場安不安全
L 線對地
✅ 127V 或 220V
⚠️ 電壓偏低 → 線路問題
N 線對地
✅ 0～3V
🚨 超過 5V → 中性偏移，查三相不平衡或 N 線阻抗過高
PE 對地
✅ 0V
🚨 有電壓 → 可能漏電或 N/PE 混接
PE 上電流 ⚡
✅ 0A
🚨 有電流 → 有漏電或 N/PE 混接，立即停機！
L 和 N 電流差
✅ 差值 < 30mA
🚨 超過 30mA → RCBO 應跳，若沒跳 → RCBO 故障
顏色識別（台灣 CNS 規範）
● 紅色 → R 相火線
● 黑色 → S 相火線
● 藍色 → T 相火線 / 單相火線
● 白色 / 灰色 → 中性線 N
● 綠色 / 黃綠 → 地線 PE
⚡ 發現違規接法的處理流程
1. 立即停機（不是只記錄）
2. 量 PE 電流（有電流 = 混接或漏電）
3. 找 N 線斷點
4. 重新拉 N 線（不是把 PE 改粗）
5. 不可妥協（趕工不是理由）
🚫 絕對禁止的 5 件事
這五件事不管什麼理由都不能做，每一件都是在埋沒有倒數計時的炸彈
1
PE 當 N 回路用
PE 有電流 → 所有外殼帶電 → RCBO 失效 → 人碰觸電
陷阱：設備能動，看起來正常，但隨時可能出人命
2
N 線斷了用 PE 跳接補
同上，且因設備能動，危險可能持續數年無人發現
最常見的趕工惡習，最危險的隱患
3
末端把 N 和 PE 短路接在一起
N 上電流流進 PE → 外殼帶電 + RCBO 失效
違反 CNS 和 IEC，任何理由都不可接受
4
打接地棒自造 N 線
大地阻抗 10～100Ω，三相不平衡時電壓漂移，設備燒毀
「感覺上 0V 所以可以當 N」— 這是錯誤認知
5
以為 N 線 0V 就可以亂摸
N 線電壓 0V 但有完整回路電流，30mA 就能造成心室纖維顫動
電壓不是危險的，電流才是。0V ≠ 安全。
