🏠 微電網（Microgrid）
技術定義
包含分散式能源（DER）、儲能（BESS）、負載的小型電力系統，透過 PCC 連接台電大電網，也能在大電網故障時獨立運作（孤島模式）。
🏠 生活比喻：社區自備水塔
大樓平常接台北自來水（台電），但屋頂還有一個大水塔（BESS）和集雨系統（太陽能）。

颱風停水時，切斷自來水管，靠水塔撐到復水。

微電網 = 有自己水塔的建築，停電也不怕。
🟢 並網模式（正常）
PCC 開，用台電電
太陽能多餘 → 存進 BESS
離峰便宜 → BESS 充電
🔵 孤島模式（停電）
PCC 關，靠 BESS + PV
BESS 啟動 VSG 模式
自建 60Hz 頻率參考
三個核心要素
PCC（那道門） ＋ BESS（水塔） ＋ EMS（智慧管理大腦）= 完整微電網
🏝️ 孤島模式（Islanding）
🏝️ 生活比喻：荒島上的露營
颱風來了，橋斷了（PCC 斷開），你和台電隔離。

✅ 有帳篷＋糧食＋水（有 BESS ＋ VSG）→ 孤島上也能活
❌ 什麼都沒有（無儲能）→ 完蛋
⚠️ 孤島的安全威脅（為什麼要偵測）
電工去外面修電線，以為已經停電了，結果你家太陽能還在偷偷供電給那條線 → 電工觸電身亡

因此 IEEE 1547 規定：大電網斷電後，逆變器必須在 2 秒內停止輸出（除非你有合法的孤島控制設計）。
規則法（OFP/OUV）
頻率/電壓超閾值才觸發
NDZ（漏偵率）= 17.9%
有 17.9% 的孤島事件偵測不到
XGBoost（明志 3/28 突破）
特徵：freq_std_5m（5 分鐘頻率標準差）
NDZ = 0%
完全消除漏偵，Tesla 孤島充電才成為可能
🎼 Grid-Following vs Grid-Forming
🎼 生活比喻：樂手 vs 指揮
Grid-Following = 樂手：跟著指揮（台電）的節拍打，很準，但指揮消失（停電）→ 整個樂團停頓，不知道該打什麼節拍。

Grid-Forming = 指揮：自己定節拍，帶著大家走。停電時拿起指揮棒，建立 60Hz，讓其他設備繼續跟著跑。
特性 Grid-Following（樂手） Grid-Forming（指揮）
頻率來源 跟著台電的節拍 自己建立 60Hz 參考
孤島能力 ✗ 不行 ✓ 可以
控制複雜度 低（PLL 同步） 高（VSG / Droop 算法）
典型用途 一般並網太陽能 / BESS 儲能主力、V2G 雙向逆變器
明志科大的情境
平常並網 → Grid-Following（跟台電）；停電孤島 → BESS 切換到 Grid-Forming（VSG 模式），BESS 變臨時指揮。
🌀 VSG（Virtual Synchronous Generator）
🌀 生活比喻：大陀螺 vs 小陀螺
傳統火力發電機 = 幾噸重的大陀螺，轉速很難突然改變 → 這叫「慣量（inertia）」。負載突然變大，頻率緩慢降，不會瞬間崩潰。

太陽能 + BESS 的逆變器 = 小陀螺，電子設備，沒有物理慣量，負載一變輸出立刻跳 → 頻率不穩定。

VSG = 用程式碼在 BESS 裡模擬那顆重陀螺，讓頻率變化平滑。
VSG 核心公式（擺動方程式）
J · dω/dt = Tm - Te - D·(ω - ω₀)
J = 虛擬轉動慣量  ｜  D = 阻尼係數  ｜  Tm = 機械力矩  ｜  Te = 電磁力矩
VSG vs Droop Control 差別
Droop Control：有稱差，無慣量（不模擬擺動方程式）
VSG：有稱差 ＋ 有虛擬慣量（同時模擬），頻率響應更像傳統發電機
明志的 VSG 應用
3/11 停電意外 → 3/16 孤島實驗啟動 → BESS 以 VSG 模式撐住孤島 → 3/30 Tesla Model Y 在孤島模式下充電成功（V2G 前置驗證）
💰 TOU 時間電價 × V2G 套利
🚕 生活比喻：計程車尖峰加價
就像 Uber 下班時間加價，TOU 電費就是「什麼時間用電，決定你付多少錢」。白天用電高峰貴、半夜便宜。
時段 夏月（6–9 月） 非夏月（10–5 月）
尖峰（平日 7:30–22:30） 6.99 元/度 4.78 元/度
離峰（其餘時間） 1.96 元/度 1.85 元/度
💡 V2G 套利計算（非夏月）
半夜 1.96 元 充電 → 下午 4.78 元 放回電網
淨收益 = 4.78 − (1.96 ÷ 0.90效率) = 2.60 元/度
一台車 30 kWh 可用容量 → NT$78 / session
真實案例（澳洲）
車主在電價飆到每度 20 澳元時，兩小時內放電賺了 560 澳元。平均每年可省/賺約 1,000 澳元。（來源：ARENA 2024 V2G 研究）
🚪 PCC（Point of Common Coupling）
🚪 生活比喻：社區大門電錶
整個校園微電網和台電之間，只有一個正式的接線點，就是 PCC。

門開（PCC 閉合）= 並網模式，台電供頻率參考，自由交換電力
門關（PCC 斷開）= 孤島模式，校園靠自己撐

門的開關 = 防孤島保護繼電器（Anti-Islanding Relay）
🟢
PCC 閉合（並網）
台電 → 提供 60Hz 頻率參考
逆變器 → Grid-Following 模式
雙向電力交換 → 可售電、可買電
🔴
PCC 斷開（孤島）
台電訊號消失
BESS 切換 → Grid-Forming（VSG）
自建頻率，校園自給自足
PCC 相關的電力指標
在 PCC 量測：有效功率（kW）、無效功率（kVAR）、功率因數（PF）、電壓諧波（THD）→ 這些是校園 EMS 管理的核心數據。
