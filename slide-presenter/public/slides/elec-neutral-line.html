Neutral Wire — Deep Dive
N 線電壓是 0V，但不能亂摸。
N 線是變壓器「自己製造」出來的，不是從外面接進來的。
N 線接地的目的，不是讓它變 0V，而是把 0V 釘住。
N 線的起源 三相電壓加總 = 0 中性點漂移 為何接大地
🔬 N 線從哪裡來？
N 線是變壓器低壓側「自然誕生」的，不是從外面接進來
製造過程（Step by Step）
1
台電高壓電（22kV）進入變壓器
高壓側採 Delta 接，沒有中性點
2
透過鐵芯磁場感應到低壓側繞組
3
低壓側三個繞組採 Star（Y）接法
三個線圈的「尾端」全部接在同一點
4
三相電壓互差 120°，任何瞬間
R + S + T = 0（永遠成立）
5
這個匯聚點的電位天然是 0V
→ 從這裡引出的導線就是 N 線
三相電壓加總 = 0（數學證明）
R = 127 × sin(ωt) S = 127 × sin(ωt + 120°) T = 127 × sin(ωt + 240°) R + S + T = 0（任何瞬間永遠成立） 因此： • 高壓側（Delta）→ 沒有 N 線 • 低壓側（Star）→ 三尾端匯聚點 = 0V • N 線在低壓側第一次誕生
🎯 關鍵認知
高壓電線桿上只有 3 條線（3P3W），N 線是在你家巷口那根路旁變壓器的低壓側才「製造出來」的。距離你家就是巷口到你家，幾十公尺而已，跟台電在哪裡無關。
🌍 N 線為什麼要接大地？
接地不是讓 N 點變 0V——N 點本來就是 0V。接地是把 0V「釘死」。
❌ 不接地會發生什麼
三相負載不可能完全平衡，N 點電位會漂移（Neutral Shift）。

某相電壓可能從 127V 飆到
180V+
接在那相的設備直接燒毀。
✅ 接地的三個作用
1. 防止中性點漂移
大地電位 = 0V，N 點被釘死不動，各相電壓維持穩定。

2. 提供 PE 漏電電流回路
漏電 → PE → 接地棒 → 大地 → 台電接地棒 → N 點回電源。

3. 建立全球共同電位基準
大地是唯一的 0V 參考點，讓所有量測有意義。
💡 一句話記住
「接地不是製造 0V，是固定 0V」

類比：N 點是一顆氣球，天然飄在 0V。接地 = 把繩子綁在地上，讓它不隨風飄走。

如果不綁（不接地），負載一不平衡，氣球就亂飄，電壓就不穩了。
中性點漂移情境模擬
情境：三相負載不平衡，且 N 點未接地 理想狀況（平衡負載）： R 相電壓 = 127V ✅ S 相電壓 = 127V ✅ T 相電壓 = 127V ✅ 實際情況（不平衡 + 未接地，N 點漂移 +50V）： R 相電壓 = 177V ❌ → 該相設備超壓燒毀 S 相電壓 = 127V ✅ T 相電壓 = 77V ⚠️ → 該相設備電壓不足，故障 → 有接地：N 點被鎖在 0V，各相電壓維持 127V ✅
🔄 N 線的完整回路路徑
電流從哪裡出發，就要從哪裡回來——閉合迴路是電力的鐵律
N 線電流完整路徑
電源（變壓器中性點） ↓ N 線沿電線桿（架空銅導線，阻抗 ≈ 0Ω） → 你家進戶盤 N 端子 → 插座 N 孔 → 負載（電流在此做功 → 熱/光/機械） ↓ 電流做完工，電壓降為 0V → 插座 N 孔（回程） → 進戶盤 N 端子 → N 線沿電線桿回去 → 變壓器中性點（閉合迴路完成） 距離：你家 ↔ 巷口變壓器 = 約 30～100 公尺
⚡ N 線的電氣特性
• 電流大小：與 L 線完全相同
• 電壓：理論 0V，實際有 1～5V 偏移
• 線徑：通常與相線相同（特殊場合需加大）
• 阻抗：越低越好，阻抗高 → 偏移電壓增大
• 色標（台灣 CNS）：白色 / 灰色
⚠️ N 線的特殊情況：諧波
三相系統中，基波（50Hz）電流在 N 線相消 = 0。
但 3次諧波（150Hz）三相同相位，不相消，反而疊加！

→ 大量非線性設備（電腦、VFD）的環境
→ N 線電流可達相線的 1.5～1.7 倍
→ 舊建築 N 線線徑不足 → 過熱風險
🏭 N 線在特殊場景的注意事項
太陽能系統、VFD 設備、辦公大樓各有不同的 N 線問題
☀️ 太陽能 / 逆變器系統
• 逆變器輸出側通常是 3P4W，N 線必須從逆變器 N 端子接，絕對不能拿 PE 代替
• 並網點的 N 線要與台電 N 線同電位，否則中性偏移 → 某相電壓異常 → 逆變器保護跳脫
• 逆變器本身會產生 3次、5次、7次諧波，N 線設計要考慮諧波電流疊加
• 太陽能工地最常見：N 線接地棒沒打好，導致接地電阻過高，中性漂移
⚙️ 變頻器（VFD）設備
• VFD 為非線性負載，產生大量諧波電流（主要 5次、7次）
• N 線諧波電流不相消，大量 VFD 並聯時 N 線電流可能是相線的 1.3～1.5 倍
• 解法：N 線線徑加大（1.5 倍）、安裝主動式濾波器（APF）
🖥️ 辦公大樓 / 資料中心
• 電腦電源供應器（SMPS）是典型三次諧波來源
• 辦公大樓 N 線電流測量常見 超過相線電流
• 舊建築 N 線可能只有 2.5mm²，實際需要 4mm² 以上
• 資料中心標準：N 線配置為相線的 2 倍截面積
🔑 現場判斷 N 線健康的方法
用鉗型電流表量 N 線電流：正常 ≤ 相線電流；超過相線電流 = 諧波問題需處理。 同時量 N 線對地電壓：正常 ≤ 3V；超過 5V = 中性偏移，查三相不平衡或 N 線阻抗過高。
🚫 能不能自己「造」中性線？
四種情境分析，只有一種合法
❌ A. 拿 PE 當 N 用
絕對不行
PE 阻抗高（銅線 + 大地路徑）
電流讓 PE 帶電 → 所有外殼帶電
三相不平衡時電壓不穩
違法且危險，RCBO 保護失效
❌ B. 打接地棒自造 N 線
不行
大地阻抗：10～100Ω
真正 N 線阻抗：接近 0Ω
三相不平衡時中性點漂移
某相電壓飆到 180V+ → 設備燒毀
違法
✅ C. 裝隔離變壓器（唯一合法）
1:1 隔離變壓器，輸出側 Star 接，中間抽頭接地
合法製造自己的中性點
醫院隔離電源、船舶電力採用此方式
需要專業設計和 CNS 認證
⚠️ D. TN-C-S 系統（特殊合法）
電源到進戶盤這段 N+PE 合一（PEN 線）
進入建築後在 MDB 強制分離
分離後永遠不能再合接
符合 CNS 及 IEC 60364 規範
N 線自造的核心判斷標準
N 線的本質 = 電壓真正穩定在 0V 的低阻抗點 條件 1：阻抗必須接近 0Ω（大地阻抗 10～100Ω，完全不符） 條件 2：必須由變壓器繞組產生（三相電壓加總 = 0 的天然中性點） 條件 3：必須接地固定電位（釘住，不讓它飄） → 「感覺上 0V 所以可以當 N」是最危險的錯誤認知
